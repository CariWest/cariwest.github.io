Common and Useful Commands

Starting Your Repository

  • init: creates an empty git repository
  • clone: clones an existing repository into a new directory

Adding Changes to Your Repository

  • add: adds file contents to the index
  • commit: record changes to a repository
  • push: updates remote references along with associated objects

Fetching & Merging Repository Changes

  • fetch: downloads objects & references from another repository
  • merge: joins two or more development histories together
  • pull: downloads objects & references from another repository; joins the development histories together.

Managing Remote Repositories

  • remote: views & manages a set of tracked repositories
  • remote add: adds a branch of a tracked repository
